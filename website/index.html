FAQ

자주 묻는 질문

인터넷이 없어도 사용할 수 있나요?＋

네. 프로젝트 작성, 검산, Excel과 영수증철 생성은 모두 로컬에서 작동합니다. 새 버전 확인과 다운로드에만 인터넷이 필요합니다.

앱을 다시 설치하면 프로젝트가 사라지나요?＋

아니요. 사용자가 선택한 위치의 .barun 파일은 앱 설치 영역과 분리되어 있습니다. 새로 설치한 앱에서 같은 파일을 열면 됩니다.

한글 Excel과 영수증철 양식이 바뀌지는 않나요?＋

제공된 공식 템플릿 사본을 읽어 결과를 만들며 템플릿 자체에는 쓰지 않습니다. 금전출납부는 기존 서식을 유지하고 필요한 행만 같은 서식으로 늘립니다.

새 버전은 어떻게 설치하나요?＋

앱이 GitHub Releases에서 새 버전을 확인해 알려줍니다. 프로젝트와 업데이트 전 백업을 저장한 뒤 서명이 확인된 파일만 설치합니다.
