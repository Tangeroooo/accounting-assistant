FAQ

자주 묻는 질문

인터넷이 없어도 사용할 수 있나요?＋

네. 웹앱을 한 번 정상적으로 연 뒤에는 필요한 파일을 브라우저에 캐시해 오프라인에서도 프로젝트 작성, 검산, Excel과 영수증철 생성을 할 수 있습니다.

브라우저 데이터를 지우면 프로젝트가 사라지나요?＋

브라우저 자동 복구본은 지워질 수 있으므로, 중요한 시점마다 .barun 백업을 내려받으세요. 백업 파일에는 모든 이미지가 포함되어 어떤 지원 브라우저에서도 다시 열 수 있습니다.

한글 Excel과 영수증철 양식이 바뀌지는 않나요?＋

제공된 공식 템플릿 사본을 읽어 결과를 만들며 템플릿 자체에는 쓰지 않습니다. 금전출납부는 기존 서식을 유지하고 필요한 행만 같은 서식으로 늘립니다.

웹앱은 어떻게 업데이트되나요?＋

페이지를 다시 열면 배포된 최신 버전을 자동으로 받습니다. 업데이트 전에 중요한 프로젝트를 .barun 파일로 한 번 백업하면 더 안전합니다.

설치형 앱에서 보안 경고가 나오면 어떻게 하나요?＋

페이지 위쪽의 작은 Windows·macOS 설치형 앱 항목을 펼치면 Gatekeeper와 Windows SmartScreen 대응 순서를 확인할 수 있습니다. 운영체제의 보안 기능 전체를 끄지 마세요.
